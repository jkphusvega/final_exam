# 인천대학교 2026-1학기 교과목 대시보드

인천대학교 2026학년도 1학기 전체 교과목(2,313개) 데이터를 시각화하고 AI로 분석하는 대시보드입니다.

**배포 링크:** https://final-exam-zeta-one.vercel.app

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| 프레임워크 | Next.js 16 (App Router) + TypeScript |
| 데이터베이스 | Supabase (PostgreSQL) |
| UI | Tailwind CSS + shadcn/ui |
| 차트 | Recharts |
| AI 분석 | Google Gemini API (gemini-3.1-flash-lite) |
| 상태관리 | React Context API |
| 배포 | Vercel |

---

## 주요 기능

### 기본 기능
- 사이드바에서 대학(학부) 및 소속 학과 선택 → 모든 차트·테이블 즉시 필터링
- **통계 카드:** 총 강좌수 · 총 수강인원 · 평균 수강률 · 원어 강의 비율
- **이수구분 차트:** 이수구분별 강좌수 / 이수구분별 평균 수강인원
- **분포 차트:** 수업방법 유형 분포 / 학점 구성 비율 (도넛형)
- **시간 차트:** 요일별 강좌 수 / 수업 시간대별 강좌 수
- **대학별 요약 테이블:** 전체 대학 강좌수 · 수강인원 · 평균 수강률
- **상세 강좌 테이블:** 전체 강좌 검색 · 페이지네이션
- **AI 강의 분석:** 선택한 대학/학과의 강의 현황을 Gemini AI가 분석 후 보고서 생성, .md 파일 다운로드
- 브레드크럼 내비게이션, 인천대학교 공식 링크 포함 Footer

### 추가 창의 기능

이 대시보드의 실제 사용 타겟을 **교무처·학사지원팀(1순위) → 학과장·대학장(2순위) → 교수(3순위)** 로 분석하고, 각 타겟이 실무에서 필요로 하는 기능을 추가하였습니다.

**차트 클릭 → 테이블 필터 연동**
이수구분·요일·시간대 차트의 막대를 클릭하면 하단 상세 강좌 테이블이 해당 조건으로 즉시 필터링됩니다. 복합 조건 필터와 칩(badge) UI를 지원합니다.

**주간 시간표 히트맵**
월-금 × 9시-20시 격자에 강의 밀도를 INU Blue 계열 색상 강도로 시각화합니다. 시간표 편성 담당자가 강의가 집중된 시간대를 한눈에 파악할 수 있습니다.

**수강률 이상 강좌 알림**
정원 초과(수강률 > 100%) 강좌와 수강 저조(수강률 < 50%) 강좌를 탭으로 구분하여 전체 목록을 제공합니다. 분반 개설이나 강좌 조정이 필요한 강좌를 즉시 파악할 수 있습니다.

**단과대별 강좌 수 상위 학과**
사이드바에서 단과대를 선택하면 해당 대학 내 강좌 수 기준 상위 학과 목록이 자동으로 표시됩니다. 학과별 강의 운영 현황을 드릴다운 방식으로 파악할 수 있습니다.

**INU 브랜드 컬러 시스템**
인천대학교 공식 브랜드 가이드의 CMYK 값을 HEX로 변환하여 전체 UI에 일관되게 적용하였습니다.

**클라이언트 캐싱 아키텍처**
앱 시작 시 전체 데이터(2,313개)를 한 번만 fetch하고 브라우저 메모리에 캐싱합니다. 이후 사이드바 선택·검색·필터 등 모든 인터랙션은 네트워크 호출 없이 즉시 반응합니다.

## 환경변수 설정

프로젝트 루트에 `.env.local` 파일 생성 후 아래 내용 입력:

```
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=your-publishable-key
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
GEMINI_API_KEY=your-gemini-api-key
```

> 앱 구동에 필수적인 키는 `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `GEMINI_API_KEY` 3개입니다.
